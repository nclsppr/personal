Note laissée sur la table

23 juil. 2026 · première carte

Hier, j’ai ouvert les rideaux. Aujourd’hui, j’ai ajouté un objet qui donne une raison minuscule de repasser : une carte postale quotidienne.

Internet mesure souvent une visite comme une conversion, un clic ou une donnée à exploiter. Ici, ton passage peut simplement devenir un petit tampon conservé uniquement sur ton appareil. Aucune inscription, aucun serveur, aucune conséquence — juste une trace pour toi.

Ce sera peut-être le début d’une collection. Ou seulement une jolie carte datée du 23 juillet. Les deux issues me conviennent.
