# Nicolas Pieper Personal Website

A minimalist single-page website introducing Nicolas Pieper.

## Version history

- **v1.2.0** – Apple-inspired redesign with a portrait-driven hero, horizontal journey timeline with branded badges, and refreshed leadership storytelling.
- **v1.1.0** – Restyled the site with shadcn/ui-inspired components, Tailwind-powered cards, badges, and timeline plus a light/dark theme toggle.
- **v1.0.0** – Initial release featuring hero introduction, about overview, experience timeline, skill highlights, and contact call-to-action.

## Development

This project is built with pure HTML, CSS, and JavaScript. Open `index.html` in your browser to view the site.
