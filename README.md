# Nicolas Pieper – Présentation WebAwesome

Cette page est une vitrine personnelle construite entièrement avec les composants [WebAwesome](https://webawesome.com/). Elle
propose un aperçu du positionnement, des valeurs et des offres d'accompagnement de Nicolas Pieper dans des contextes produits &
tech régulés.

## Aperçu du contenu

- **Héros immersif** : badges de positionnement, storytelling, call-to-action et appel à la co-création.
- **Repères chiffrés** : cartes WebAwesome pour visualiser expérience, équipes accompagnées et garanties de conformité.
- **Focus d'accompagnement** : trois cartes détaillant les leviers d'intervention prioritaires.
- **Expérience récente** : cartes chronologiques illustrées avec les logos disponibles dans `images/`.
- **Mode de collaboration** : groupe d'onglets (`wa-tab-group`) décrivant cadence, communication et mesure d'impact.
- **Racines** : accordéons (`wa-details`) couvrant formation, mentorat et veille.
- **Call-to-action final** : carte pleine largeur invitant au contact.

## Visualiser la page

Ouvrez directement `index.html` dans l'aperçu de votre IDE (par exemple **Open Preview** dans VS Code / Codex) pour découvrir la
page. Aucun serveur n'est requis : toutes les dépendances WebAwesome proviennent du CDN officiel.

## Personnalisation rapide

- Les images sont stockées dans `images/` et peuvent être remplacées pour personnaliser le rendu.
- La palette et les espacements reposent sur les tokens WebAwesome (`--wa-*`). Ajustez-les via la feuille de style inline dans
  `index.html`.
- Les sections sont écrites en français ; adaptez facilement à d'autres langues en modifiant les textes des composants.
